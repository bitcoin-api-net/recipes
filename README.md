# Bitcoin API Recipes

Official code examples and recipes for integrating with [Bitcoin API](https://bitcoin-api.net).

## Structure

Each subdirectory in this repository is a standalone recipe/example:

- `show-current-price`: A simple dashboard card showing real-time Bitcoin price.

## Usage

Each recipe has its own `README.md` with specific instructions. Generally:

1. Enter the recipe directory.
2. Run `npm install`.
3. Run `npm run dev`.
